## Acceptance criteria

### AC1 — Globe fills area below header to tab bar, no top clip, no page scroll — PASS
- **1280×900:** canvas `top=69` == header `bottom=69` (no clip); canvas
  `672×828`, centered (`left=304`, `right=976` → symmetric 304px margins);
  `scrollHeight=900 == innerHeight=900` → no vertical page scroll.
- **390×844:** canvas `top=69` == header `bottom=69` (no clip); canvas height
  `708`, bottom `777`; tab bar `top=784.5` → globe sits above the tab bar, not
  hidden behind it; `scrollHeight=844 == innerHeight=844` → no page scroll.
- Globe large + centered in both. Evidence: `shots/s3-travel-light-1280.png`,
  `shots/s3-travel-light-390.png`, `shots/s3-desktop-dialog.png`.

### AC2 — Theme-aware category legend, swatches match fills, corner, no collision — PASS
- Legend overlay present top-left (1280: `left=320,top=85`, compact `84×78`),
  `pointer-events:none` (non-blocking), 3 rows Home/Lived/Visited with colored
  swatch dots.
- **Legend swatch == fill color (pixel-sampled):**
  - Home: legend swatch `rgb(220,38,38)`; India fill pixel `rgb(217,41,50)` →
    match (Δ≈12.7, anti-alias/opacity). India tap confirms Home.
  - Lived: legend swatch `rgb(37,99,235)`; US fill pixel `rgb(51,109,213)` →
    same blue family (US was emphasized/lightened when sampled).
  - Visited: legend swatch `rgb(22,163,74)` (== `#16a34a` exactly); Japan/Costa
    Rica render green (visible in `shots/s3-india-search.png`).
- **Theme-aware:** legend swatches re-read per theme — light `rgb(220,38,38)/
  (37,99,235)/(22,163,74)`; dark `(248,113,113)/(96,165,250)/(74,222,128)`;
  sunset `(185,28,28)/(29,78,216)/(21,128,61)` — each exactly the theme `cat`
  values. Legible in all 3 (screenshots).
- **No collision (mobile):** legend top-left; sheet bottom-anchored (`top=686`)
  → no overlap with sheet or tab bar. Evidence: `shots/s3-mobile-sheet.png`,
  `shots/s3-travel-dark-1280.png`, `shots/s3-dialog-sunset.png`.

## Soft notes (not failing)

- The active-country emphasis uses `lighten(color, 0.4)`, which on light theme
  desaturates the active fill noticeably (US looks pale lavender when open in
  `s3-desktop-dialog.png`). It reads as "the focused one" and differs clearly
  from resting, so it passes, but a brighter/more-saturated emphasis (rather than
  lightening toward white) would read more confidently as the active country.
- France renders its overseas territory (French Guiana) as a green sliver near
  South America in addition to metropolitan France. This is explicitly allowed by
  SPEC §4.1 (acceptable; optional polish to filter to the largest ring).
- Desktop dialog and mobile sheet both use the same component breakpoint-switched;
  the desktop dialog is keyboard-accessible (focus enters on open, Escape closes).
  Focus-return-on-close was not separately asserted here but Escape/close work and
  no focus trap was observed.
