# Kandou

A volunteering app (iOS & Android, built in React Native) connecting people
with local opportunities, plus CSR management tools for organizations.
Reached 600 users across 17 San Francisco nonprofits, facilitating 25
volunteer events and 125+ community service hours.

Source lives in the private repos `kandou-app-legacy` (mobile) and
`kandou-web-legacy` (web + server). Screenshots kept here for posterity.
